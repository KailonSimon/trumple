# Trumple

Clone of [Wordle](https://www.nytimes.com/games/wordle/index.html)
Made using React.

[**Live Link**](https://trumple.app/)

### To Run Locally:

Run the following commands:

```bash
$> git clone https://github.com/jamesurobertson/trumple.git
$> cd trumple
$> npm install
$> npm run start
```
